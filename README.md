## Tech Stack

**Client:** Ruby 2.7.6

**Testing:** RSpec


## Runing tests Locally

Clone the project

```bash
  git clone git@github.com:vmartin/cart_scan.git
```

Go to the project directory

```bash
  cd cart_scan
```

Install dependencies

```bash
  bundle install
```

Run all tests

```bash
  ./execute_all_rspec_test.sh
```
